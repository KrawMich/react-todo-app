# React Todo App

Simple todo application built with React and TailwindCSS.

## Live Demo

[Open app](https://react-todo-app-gray-xi.vercel.app/)

## Features
- add tasks
- edit tasks
- delete tasks
- mark tasks as completed
- filter tasks (all / active / done)
- drag & drop reordering
- localStorage persistence

## Tech stack

- React
- Vite
- TailwindCSS
- dnd-kit

## Installation

npm install  
npm run dev
